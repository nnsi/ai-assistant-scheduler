# ブラウザテスト

browser-testerサブエージェントを使ってブラウザでの動作確認を行う。

## 使い方

```
/browser-test ログイン画面が正しく表示されるか確認
/browser-test スケジュール作成フローをテスト
/browser-test http://localhost:5173 でトップページを確認
```

## 実行手順

1. **browser-testerサブエージェントを起動する**

以下のようにTaskツールでbrowser-testerを起動すること：

```
Task tool:
- subagent_type: browser-tester
- prompt: [ユーザーの確認内容をそのまま渡す]
```

2. **開発サーバーが起動していることを確認**

ブラウザテスト前に開発サーバーが起動している必要がある。
ユーザーに確認するか、`http://localhost:5173` へのアクセスを試みる。

## 確認できること

- ページの表示確認
- フォーム入力・送信
- ボタンクリック・ナビゲーション
- コンソールエラーの確認
- スクリーンショット取得
- 複数タブの操作

## 注意事項

- 開発サーバーはユーザーが起動する（自分で起動しない）
- テスト後はブラウザを閉じる
- エラーが出たらコンソールメッセージを確認する
